Experimental Mode
Please be advised that we are not reliable for any side effects occur while you try experimental settings.

Also take care to deactivate every experimental setting before disabling 'Experimental Mode'. Just disabling this option, will not return your devices to the default configuration! (e.g. if you enable rtsp streaming on a doorbell, disable it on the device settings before disabling 'Experimental Mode').
When enabled, the plugin allows you to change specific behaviour of some eufy devices. For example you can try to enable RTSP streaming on Doorbells (which usually don't support RTSP streaming).
